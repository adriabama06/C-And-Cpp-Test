# Formats
## For `printf()` and `scanf()`

```
int	                    %d, %i
char	                %c
float	                %f
double	                %lf
short int	            %hd
unsigned int	        %u
long int	            %li
long long int	        %lli
unsigned long int	    %lu
unsigned long long int	%llu
signed char         	%c
unsigned char	        %c
long double  	        %Lf
```
